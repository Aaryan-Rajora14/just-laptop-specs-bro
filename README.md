# just-laptop-specs-bro 💻

> "Hey what specs does your new laptop have?"
> "Give me an hour."

My friend **Rohan** (or whoever) texted asking about my new Acer.
Normal people send a screenshot. I spent an hour building a full interactive
HTML system profile with animations, a port diagram, a performance page,
and a links tab with every product image from Amazon.

He just wanted to know the RAM.

---

## 🖥️ What's Inside

A single `laptop_profile.html` file you open in any browser — no frameworks,
no build step, no dependencies. Just vibes and `Orbitron` font.

**4 pages:**
- **Overview** — all specs, GPU breakdown, display, inputs, system info
- **Ports & I/O** — SVG port diagram, left/right side breakdowns, wireless
- **Performance** — benchmark bars, gaming FPS estimates, full spec table
- **Links** — Amazon buy links, product gallery, image CDN URLs

---

## 🚀 Usage
```bash
git clone https://github.com/YOUR_USERNAME/just-laptop-specs-bro.git
cd just-laptop-specs-bro
# open laptop_profile.html in your browser. that's it.
```

Or just [download the HTML file](./laptop_profile.html) and double-click it.
